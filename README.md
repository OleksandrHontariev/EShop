# EShop

EShop is a simple and scalable e-commerce platform designed for managing products and handling purchases.

The system allows users to create, update, and manage an unlimited number of products, making it suitable for small to large-scale online stores.

## Features
- Product management (create, edit, delete)
- Scalable architecture for large product catalogs
- Designed for future purchase and order functionality

## Purpose
This project is built for learning and practicing backend development, including working with databases, APIs, and Entity Framework.

## Tech Stack
- C#
- .NET
- Entity Framework
